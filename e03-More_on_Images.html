3. More on images¶

  1. Import the numpy and skimage packages

  1. Using the skimage import function, load the image visible at this link: http://wiki.cmci.info/sampleimages/cells_Actin.tif and put it in a variable called image

  1. Check what are the dimensions of the image

  1. Check what is the image type. Is it an 8 or 16 bit image ?

  1. Plot the image using Matplotlib (that you have to import). Change the color map to ‘gray’ and make the image larger

  1. Plot a histogram of the pixel values. Change the bin size to better see the distribution

3b. More on images: cropping, logical indexing¶

  1. Make a cropped version of the image by selecting 100 rows for which you want all columns and plot it.

  1. Select a region in the image of 100x100 pixesl and create a cropped image. Plot it.

  1. Using this second cropped image, create a boolean mask by finding all pixels above some reasonable threshold (use the histogram above to find a value)

  1. Plot your mask

  1. Get all the values of all the pixels which are selected in the mask and store in a variable

  1. Plot a histogram of all the values
